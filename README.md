# Auto-Complete Text Editor

Trie-based autocomplete and spell-checking system with heap-based ranking and prefix caching.

## Features
- Trie-based prefix search
- Min-Heap top-K ranking
- Circular buffer prefix cache
- Frequency-based learning
- Client-server architecture in C

## Tech Stack
- C (Backend)
- HTML/CSS/JavaScript (Frontend)
- Winsock (Networking)
